# Task 2:  Sentiment Analysis using NLP Pipeline  &  ML Models  

##  Overview

The objective is to build an end-to-end Sentiment Analysis system using NLP techniques and Machine Learning models.

---

##  Pipeline

Raw Text → Preprocessing → Feature Engineering → Model Training → Evaluation → Comparison

---

##  Features

* Text preprocessing (cleaning, stopwords, lemmatization)
* Feature extraction using BoW and TF-IDF
* Model training using:

  * Logistic Regression
  * Naive Bayes
  * Decision Tree
* Model evaluation using:

  * Accuracy
  * Precision
  * Recall
  * F1 Score

---

##  Dataset

IMDb Movie Reviews Dataset (Kaggle)

---

##  Conclusion

Logistic Regression with TF-IDF provided the best performance for sentiment classification.
